```json

    "@type": "Question",

    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Cyprus fast-track residency takes 2 months..."

  }]
```

#### HowTo Schema
```json
{
  "@type": "HowTo",
  "name": "How to Apply for Dubai Golden Visa",
  "step": [
    {"@type": "HowToStep", "text": "Prepare documents..."},
    {"@type": "HowToStep", "text": "Submit application..."}
  ]
}
```

```json
{
```
