## 🛠️ Tech Stack

- **Framework**: Astro 5.x + React + TypeScript
- **Styling**: Tailwind CSS 4.x
- **CMS**: Sanity Universal (93ewsltm/production)
- **Images**: Flux Pro via Replicate
- **Deployment**: Vercel
